What is x402

Payment
Required.

RFC 7231 reserved HTTP status code 402 for future use — a placeholder for a payments layer that never arrived. x402 fills that gap on the Stellar blockchain: a fixed-supply token purpose-built for frictionless, programmable payments between people, services, and machines.


No inflation. No new supply. Ever. The issuer account is locked after genesis, making the 8 billion unit cap a cryptographic guarantee, not a promise.
